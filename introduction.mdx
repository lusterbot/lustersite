Luster is a Discord bot built for server owners and moderators who want one tool that handles everything. From keeping your server safe with bans, mutes, and AntiNuke protection, to building community through leveling systems, economies, and Last.fm integration — Luster brings it all together under a single, consistent command interface. You don't need to juggle multiple bots or complicated dashboards; everything runs through straightforward prefix commands that your whole team can learn quickly.

## Command categories

Luster organizes its commands into focused categories. Run `,help` at any time to see the full list, or run `,help <category>` to explore a specific area:

| Category | What it covers |
|---|---|
| `utility` | General-purpose utility commands |
| `fun` | Games and entertainment |
| `mod` | Moderation and server protection |
| `social` | Social interaction between members |
| `economy` | Currency, jobs, shop, and trading |
| `premium` | Exclusive features for premium users |
| `server` | Server setup and configuration |
| `FM` | Last.fm music integration |
| `pokemon` | Pokémon catching and collection |

<Tip>
  Run `,help <category>` to browse commands inside any category, or `,help <command>` for detailed usage of a specific command.
</Tip>

## Default prefix

Luster's default command prefix is `,` (a comma). For example, to check the bot's latency you would type `,ping`. You can change this prefix per-server — see [Configure Luster](/configuration) for instructions.
